# 兼容性矩阵

SDK 版本：`0.1.0-alpha.1`。这里的“通过”表示仓库存在独立测试 BDA 和可复核结果，
不表示所有参数组合、固件版本或异常路径均已覆盖。

## 主机与目标

| 项目 | 当前基线 | 状态 |
|---|---|---|
| 构建主机 | Windows PowerShell，Python 3.10+ | CI 覆盖 |
| 编译器 | `mipsel-none-elf-gcc` 15.2.0 | 构建覆盖 |
| 目标 CPU | MIPS32 little-endian | 已确认 |
| 目标固件 | BBK 9588 `kj409588` / C200 | 主要目标 |

| 真机 | BBK 9588 C200 | 部分 API 验证 |

## 稳定性等级

- **公开**：位于 `sdk/include/`，存在动态测试与开发文档。
- **模拟器验证**：只承诺表中固件模拟环境已观察到预期行为。
- **真机验证**：至少完成启动、目标操作、清理并返回菜单闭环。
- **研究**：位于 `reverse/`；ABI、命名和生命周期可随证据变化。

真机失败、死锁或资源泄漏会覆盖较早的模拟器成功结论。需要扩大兼容范围时，请提交
固件标识、测试 BDA、源码、完整实时日志和退出后的系统状态。
